Banana Shooter Documentation
===================================

This document is designed to make `Banana Shooter <https://store.steampowered.com/app/1949740/Banana_Shooter/>`_'s modding features easier to understand

What This Covers
----------------------------
These docs cover the following aspects:

	1. How to make a proper map and publish it for everyone to play **(workshop map also have chance to be on the official servers)**.
	
	2. How to master the map editor using custom objects.
	
	3. How to host a community server which anyone can play on **(Both Linux and Windows)**.

Disclaimer
---------------------------
The map editor is still in beta so it might be a little buggy but the most part of it is working great. If you find any issues please report them in the `Official Discord Server <discord.gg/bananashooter>`_

Contributing
----------------------------------

Anyone can contribute towards the *Banana Shooter* modding documentation! To make any additions or changes, `fork the GitHub repository <https://github.com/CodingDaniel1/bs-docs/fork>`_. To submit an issue, go to the `the GitHub issue page <https://github.com/CodingDaniel1/bs-docs/issues>`_.
